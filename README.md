# PP2 Practice 1

This repository contains Python practice examples for PP2 course.

## Files

- hello_world.py — prints "Hello, world!"
- variables.py — shows variables and prints them
- data_types.py — shows different simple types: int, float, bool, str
- casting.py — shows type casting (convert values)
- strings.py — shows string methods: .upper(), .lower()

## Notes

Simple types store single values.  
Complex types can store multiple values, like lists or dictionaries.  

All code is in Practice1/python-basics/ and can be run with Python 3.
