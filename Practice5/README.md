# Practice 5 – Python Regular Expressions

This practice demonstrates the use of Python regular expressions using the re module.

Files in this folder:

regex_exercises.py  
Contains solutions to 10 regular expression exercises such as pattern matching, replacing characters, splitting strings, and converting naming styles.

receipt_parser.py  
Parses a pharmacy receipt from raw.txt using regular expressions. The program extracts:
- product names
- prices
- total amount
- date and time
- payment method

The extracted data is printed in structured JSON format using the json module.
